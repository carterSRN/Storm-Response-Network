## Preview

Open `index.html` in a browser.

## Publish on GitHub Pages

1. Create or open the SRN GitHub repository.
2. Upload `index.html`, `styles.css`, and `script.js`.
3. Open repository **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose the `main` branch and `/ (root)`.
6. Save.

## Important

The site intentionally describes SRN as an independent community network and avoids implying that it is a government agency, public-safety department, or official arm of NOAA, NWS, FEMA, law enforcement, fire, EMS, or emergency management.
